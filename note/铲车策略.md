# plan

我将写新功能给铲车。铲车没有舵机结构，他就是普通塑料铲子在前面，所以这个车基本就是控制路线。
现在已经完成了基础定位，基础路径规划，以及路径规划出来的路径转译到route函数的功能。现在我将写一些具体的策略，来让铲车在比赛中有更好的表现。

我希望小车的表现是，在常见路径区域巡逻，走过一段时间后回到得分区，然后再巡逻，来回循环。这样可以保证在比赛中有较高的得分效率。需要引出定义，就是哪里是自家得分区，然后地图可移动区域的定义，部分单边障碍已经在astar实现。
