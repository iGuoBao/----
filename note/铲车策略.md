# plan

始终遵守claude.md中的原则

我将写新功能给铲车。铲车没有舵机结构，他就是普通塑料铲子在前面，所以这个车基本就是控制路线。
现在已经完成了基础定位，基础路径规划，以及路径规划出来的路径转译到route函数的功能。同时已经有了最基本的随机选点，导航到点，并有条件的返回得分点。有简单的卡住检测，基于时间超时检测。现在我希望规划一个完整的agv流程，特别是自恢复定位，自恢复任务。

当前小车硬件有以下功能
1. 双轮电机，差速转向。
2. 基础姿态传感器，可返回姿态角度，加速计，陀螺仪数据。
3. 轮子强绑定的编码器，可以返回轮子转动的距离。

希望基于以上硬件，完成agv流程，特别是自恢复定位，自恢复任务。
先计划。

小铲车在栅格地图里，这是个比赛地图。双方会可能发生碰撞，所以我期望小车在这个环境里有鲁棒性表现，如同agv一样，即使碰撞丢失定位，也能通过自恢复定位，继续完成任务。
